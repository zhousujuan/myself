外贸获客工具任务需求清单

来源：2026年5月19日外贸获客工具沟通会议记录

编号 优先级 模块 需求/问题 会议反馈要点 建议处理方式 状态
REQ-001 P0 AI 获客 提升潜客匹配精准度 出现品类、材质、行业不匹配的客户，如不锈钢橱柜搜到珠宝、大牌等。 优化行业打标、关键词匹配、材质识别和大牌过滤逻辑。 待优化
REQ-002 P0 AI 获客 支持按强/中/弱匹配排序 用户希望强匹配客户排在前面，一般或弱匹配客户靠后。 在客户列表中增加匹配等级排序与筛选。 已在排版中
REQ-003 P0 AI 获客 增加“不感兴趣/移到后面”功能 用户不想关注大量客户，只想把明显不精准的客户踩下去。 增加负反馈按钮，支持移后、隐藏或降低推荐权重。 待评估
REQ-004 P0 获客任务 解决任务长时间卡住问题 部分获客任务运行一两个小时，甚至一直卡住或显示评分中。 增加任务超时机制、失败提示、重跑入口和后台监控。 待排查
REQ-005 P0 邮箱/联系人 提升邮箱准确度和有效率 部分客户没有有效邮箱，公共邮箱较多，自动回复较多。 接入新数据源，优化 KP 联系人识别和邮箱验证。 进行中
REQ-006 P1 类目选择 优化行业类目推荐 不锈钢橱柜、户外橱柜等行业找不到强相关类目。 根据产品描述自动推荐类目，减少用户手动选择成本。 规划中
REQ-007 P1 类目选择 优化类目示例说明 现有示例让用户误解，不知道该如何选择行业。 调整示例文案，增加更明确的行业路径和产品示例。 待优化
REQ-008 P1 获客流程 支持国家批量选择 用户反馈每次手动填写国家比较耗时。 增加国家批量选择、常用国家组、历史选择记忆。 待评估
REQ-009 P1 获客流程 优化同时任务数量限制 当前只能同时触达 3 个获客任务，影响效率。 增加任务队列、排队状态展示，或按账号权益提升并发数。 待评估
REQ-010 P1 潜客同步 强化潜客同步提示 用户不知道需要手动同步到潜客公海，也不清楚积分消耗。 增加明显提示、确认弹窗、积分扣除说明和同步成功反馈。 待优化
REQ-011 P1 邮件生成 提升邮件内容差异化 用户反馈邮件大差不差，主题和正文同质化明显。 增加更多邮件主题、开发角度、语气风格和场景策略。 待优化
REQ-012 P1 邮件生成 支持公共邮箱转介绍话术 部分联系人不是采购，希望能请求对方转给负责人。 增加公共邮箱/非关键联系人专用邮件模板。 待设计
REQ-013 P1 背调报告 增强报告可行动性 用户觉得报告太官方，看完仍不知道下一步怎么开发客户。 增加“开发切入点”“邮件建议”“客户痛点”“推荐话术”等内容。 待优化
REQ-014 P1 背调报告 将背调结果用于邮件生成 当前背调对邮件生成的帮助不明显。 生成邮件时自动引用背调中的客户痛点、产品匹配点和官网信息。 规划中
REQ-015 P2 图片素材 支持邮件多图上传 当前只能选一张图片，用户希望可上传多张产品或工厂图片。 支持多图选择、排序和插入位置配置。 待评估
REQ-016 P2 图片素材 支持图片分类和标签管理 图片较多时筛选困难，只能靠人工查看。 增加图片标签、分类、用途说明和搜索筛选。 待设计
REQ-017 P2 邮件触达 支持邮件签名/尾签设置 用户希望使用固定签名图片或尾签内容。 确认第三方邮件接口是否支持独立签名设置，再设计入口。 待调研
REQ-018 P2 推荐关键词 优化推荐关键词加载速度 用户点击产品推荐关键词时反馈很慢。 排查接口性能、缓存策略和网络/VPN 影响。 修复中
REQ-019 P2 反馈入口 增强意见反馈入口可见性 当前反馈入口较隐蔽，用户不容易发现。 在主要页面增加明显反馈按钮，支持截图上传和问题分类。 待优化
REQ-020 P2 客户端 上线 Mac 版获客小助手 用户询问 Mac 版什么时候上线，会议中提到预计下周。 完成 Mac 版打包、测试和发布说明。 预计下周
